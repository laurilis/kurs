Hausaufgaben

  1. Öffnen Sie den Dateiexplorer. Dieser befindet sich im Regelfall in der Taskleiste.
  2. Klicken Sie in der Navigationsleiste des Explorers (Ordnerübersicht links) auf Desktop, sodass Ihnen der Inhaltsbereich des Explorers den Inhalt des Desktops anzeigt.
  3. Erstellen Sie mithilfe des Menübands einen neuen Ordner, indem Sie zuerst auf Start klicken und dann auf “Neuer Ordner”. Nennen Sie diesen Apfel und betätigen Sie die Entertaste.
  4. Erstellen Sie einen weiteren neuen Ordner und nennen Sie diesen Ananas.
  5. Benennen Sie nun den Ordner Ananas um in Aal, indem Sie mit rechts auf den Ordner klicken und mit links auf Umbenennen klicken.
  6. Erstellen Sie einen dritten neuen Ordner und benennen Sie ihn Alge.
  7. Markieren Sie die Ordner Aal und Alge, indem Sie die STRG Taste gedrückt halten, mit der linken Maustaste auf die beiden Ordner klicken und anschließend die STRG Taste loslassen.
  8. Verschieben Sie die markierten Ordner in den Ordner namens Apfel, indem Sie die Maus auf den markierten Bereich bewegen, die linke Maustaste drücken und gedrückt halten, Sie die Maus auf den Ordner Apfel bewegen (es sollte eine blaue Markierung erscheinen) und Sie die linke Maustaste loslassen.
  9. Öffnen Sie den Ordner Apfel, indem Sie ihn doppelt mit links anklicken.
  10. Sie sollten sich nun im Ordner Apfel befinden. Sehen Sie sich die Adresszeile an. Darin sollten Sie sehen, dass der Ordner Apfel ein Ordner von Desktop ist und dieser in der Regel ein Ordner von Dieser PC ist (möglicherweise verschieden von PC zu PC). Im Inhaltsbereich sollten Sie die beiden Ordner Alge und Aal vorfinden.
  11. Klicken Sie auf Start und im Menüband, sich auf der rechten Seite befindend, auf die Option “Alles auswählen”.
  12. Verschieben Sie die beiden Ordner in den Ordner Desktop, der sich links in der Navigationsleiste befindet.
  13. Klicken Sie nun auf den Pfeil nach oben, sodass Sie eine Ordnerebene “zurück” gehen und sich anschließend wieder im Ordner Desktop befinden.
  14. Markieren Sie nun die Ordner Apfel und Aal mit gedrückter STRG Taste und Linksklick auf diese.
  15. Verschieben Sie die beiden Ordner in den Ordner Alge.
  16. Klicken Sie mit der rechten Maustaste auf den Ordner Alge.
  17. Klicken Sie mit der linken Maustaste auf die Option Löschen.
  18. Klicken Sie auf Start und sehen Sie sich in Ruhe die Bandbreite an Funktionen an, die das Menüband enthält. Klicken Sie nun auf ein beliebiges Element(Ordner oder Datei) mit der rechten Maustaste und vergleichen Sie, welche Funktionen Ihnen dieser offenbart.
  19. Fertig!








    20. Extras

    1. Bewegen Sie den Regler der Lautstärke je einmal zum linken Ende und anschließend zum rechten Ende. Deaktivieren Sie danach den Ton.
    2. Ändern Sie die Bildschirmhelligkeit nach Ihrem Belieben (Infocenter! Erweiterte Ansicht!).
    3. Öffnen Sie die Programmliste des Startmenüs und öffnen Sie das letzte Programm, das sich darin befindet. Minimieren (nicht schließen!) Sie anschließend das Programmfenster.
    4. Aktivieren Sie den Nachtmodus, der sich im Infocenter verbirgt.
    5. Verschieben Sie ein Programm, Ordner oder Datei Ihrer Wahl, die sich auf dem Desktop befindet, an den rechten Rand des Desktops.
    6. Öffnen Sie das eben verschobene Programm, Ordner oder Datei.
    7. Schließen Sie das eben geöffnete Programm, Ordner oder Datei.
    8. Verschieben das Programm, Ordner oder Datei wieder an die ursprüngliche Position.
    9. Rufen Sie das zuvor minimierte Programm auf, das sich an letzter Stelle der Programmliste im Startmenü befindet, und schließen Sie es.




    Zusatzaufgaben Sitzung 8

    1. Öffnen Sie ein neues=leeres Dokument eines Textverarbeitungsprogramms Ihrer Wahl.
    2. Geben Sie die folgenden Sonderzeichen in das Textfeld ein: / € + § *
    3. Klicken Sie auf Speichern unter. Gegebenenfalls, abhängig vom Textverarbeitungspro-gramm, klicken Sie auf Durchsuchen. (Bei Microsoft Word ist dies z.B. erforderlich, bei Word Pad nicht)
    4. Geben Sie als Dateinamen Übungsaufgabe Sitzung 8 ein.
    5. Wählen Sie in der Navigationsleiste den Desktop als Speicherort, indem Sie darauf klicken.
    6. Falls sich auf Ihrem Desktop bereits der Ordner Computerkurs befindet, speichern Sie das Dokument in diesem ab, indem Sie diesen per Doppelklick öffnen. Falls dieser nicht vorhanden ist, erstellen Sie einen neuen Ordner mit dem Namen Computerkurs, öffnen Sie diesen per Doppelklick und speichern Sie das Dokument ab.
    7. Schließen Sie das Textverarbeitungsprogramm und minimieren oder schließen Sie gegebenenfalls alle geöffneten Programme, sodass Sie sich auf dem Desktop befinden und überzeugen Sie sich von den Früchten Ihrer Arbeit: Sie sollten dort den Ordner namens Computerkurs inklusive des Dokuments vorfinden.
    8. Falls Sie den Ordner nicht benötigen, klicken Sie mit der rechten Maustaste auf den Ordner und anschließend mit der linken Maustaste auf Löschen.
